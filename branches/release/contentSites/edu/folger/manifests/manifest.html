Folger Shakespeare Library Image Masters

Manifest Page to allow preservation by LOCKSS daemons in the MetaArchive Network
LOCKSS system has permission to collect, preserve, and serve this Archival Unit.

Collection Info:

  • Conspectus Collection(s): Folger Digital Image Masters
  • Institution: Folger Shakespeare Library
  • Contact Info: Jim Kuhn
  • MetaData: MetaData Manifest

This collection consists of master digital images of Folger Shakespeare Library collection material. Each image also includes a ruler and a color correction reference card. These images have not been cropped, sharpened, color-corrected, or otherwise post-processed through image editing software. Descriptive and technical metadata for each image can be found in the accompanying database, which is present here in two formats: native FileMaker Pro (fp7) and exported xml.

LOCKSS starts crawling at this page expecting that it is called

  • manifest.html
It harvests only content/images that are referenced on manifest pages with URLs:
  • http://metaarchive.folger.edu/*/manifest-FROM*.html, where FROM is a number, and
  • http://metaarchive.folger.edu/*/manifest-FROM*-TO.html, where FROM and TO are numbers

Please add any additional images or remove references to deleted images to/from the corresponding manifest pages.

Only images referenced in a manifest page reachable from this page are preserved in the MetaArchive network. Referenced images must match the crawl pattens given in the LOCKSS plugin used for this collection; please refer to the collections page in the MetaArchive conspectus mentioned above.
