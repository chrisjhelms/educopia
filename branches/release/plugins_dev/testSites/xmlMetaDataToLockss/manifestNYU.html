NUY DSPACE Mockup Manifest Page

Manifest Page to Allow preservation by LOCKSS daemons in the MetaArchive Network
LOCKSS system has permission to collect, preserve, and serve this Archival Unit.

Collection Info:

  • Conspectus Collection(s): This demonstration site has no conpsectus entry
  • Institution: MetaArchive Org
  • Contact Info: support at metaarchive.org

This collection consists of MetaData (XML files) and document files referenced in the MetaData. A stylesheet is used on the server side to transform the XML data into HTML with links to document files, so that a LOCKSS crawl guided by a plugin can discover these document links. A corresponding LOCKSS plugin should contain rules that INCLUDE the xml (metadata) and the documents themselves.

Links for LOCKSS to start crawling

  • NYU-ArtsAndScience-Misc contains links to two record sets
  • NYU-ArtsAndScience-Sanitation
  • NYU-Stern-IOMS-IS
  • NYU-Stern-IOMS-OM

This particular example shows how one might use the OAI/DC export at http://archive.nyu.edu, new York University's DSPACE based institutional repository. This manifest page includes links to dynamically generated DublinCore XML of particular record sets. The style sheet and script on the the webserver cooperate to transform the generated Dublin Core XML to HTML with links to DSPACE item pages that end in '?mode=full'. Including this parameter in links enables us to define a simple plugin that has the usual base_url parameter and a hdl_id parameter. The folllowing rules create the desired behaviour of including metadata and docuemt files in a crawl avoiding the inclusion of DSPACE collection and community pages.

  1. EXCLUDE_NO_MATCH ^<base_url>
  2. INCLUDE manifest.html$
  3. INCLUDE set=hdl_2451_<hdl_id>
  4. INCLUDE ^<base_url>/.*mode=full$
  5. INCLUDE ^<base_url>/bitstream/.*

To use this approach to prepare a DSPACE/XML based collection for preservation with LOCKSS/MetaArchice it is easiest to place the stylesheet, transformer script, and manifest page on the same server as the content that is to be preserved.

The Details: Script, StyleSheet ...

  • transform.php, php script that applies the stylesheet
  • nyu-oai.xsl style sheet that converts DublinCore XML to html
  • http://www.eprints.org/software/xslt.php style stylesheet developed for eprints, which we used as template
